# 图片内容审查卫士

基于视觉大模型的 AstrBot 图片审核插件。插件会监听配置范围内的群聊或私聊图片，过滤 QQ 表情、贴图、GIF、APNG 等不适合审核的内容，只把普通静态图片交给视觉模型判断。

## 功能

- 按群号或私聊用户 ID 控制审核范围。
- 支持直接选择 AstrBot 已配置的视觉模型提供商。
- 支持敏感文字和禁用画面两类规则。
- 默认忽略表情包、贴图、GIF 和 APNG，不影响同条消息里的普通图片。
- 支持同条消息中的文字、表情和普通图片混发检测。
- 命中违规后可撤回群消息、禁言用户、向管理员上报证据。
- 模型返回格式异常时默认放行，避免因为解析失败误封。

## 配置

| 配置项 | 说明 |
| :--- | :--- |
| `group_scope` | 启用审核的群号列表，`["0"]` 表示所有群，空列表表示不审核群聊。 |
| `private_scope` | 启用审核的私聊用户 ID 列表，`["0"]` 表示所有私聊，空列表表示不审核私聊。 |
| `audit_provider_id` | 图片审核使用的视觉模型提供商 ID。配置页会显示“选择提供商...”。留空时使用当前会话默认聊天提供商。 |
| `custom_vision_prompt` | 额外审核要求。建议写具体画面或文字，不要写宽泛概念。 |
| `sensitive_texts` | 图片中禁止出现的文字。 |
| `forbidden_descriptions` | 图片中禁止出现的画面描述。 |
| `ignore_stickers` | 是否忽略 QQ 表情、贴图等非普通图片。 |
| `ignore_gif` | 是否忽略 GIF 和 APNG。 |
| `max_images_per_message` | 单条消息最多审核的图片数量。 |
| `check_probability` | 抽查概率，范围 0.0 到 1.0。 |
| `enable_recall` | 命中违规后是否撤回群消息。 |
| `ban_duration` | 命中违规后的群禁言时长，单位秒，0 表示不禁言。 |
| `report_target_id` | 违规报告接收目标，可填管理员 QQ 号或 AstrBot `unified_msg_origin`。 |

## 规则建议

`forbidden_descriptions` 应使用精确描述。例如需要拦截地图类图片时，写 `东亚地图` 比写 `东亚` 更稳。后者可能让模型把建筑、风景、地理联想内容误判为违规。

默认规则为空，插件不会在没有明确规则时处罚用户。启用前至少配置 `sensitive_texts` 或 `forbidden_descriptions` 其中一项。

## 安装

把插件目录放入 AstrBot 的 `data/plugins/` 目录，重启 AstrBot 后在配置页填写审核范围和模型配置。

撤回和禁言依赖 OneBot 11 协议端能力，Bot 需要在群内具备管理员权限。
